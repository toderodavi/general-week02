Week 2 - Clicker game (patting the pichu!)

This project is a clicker game made with JS, CSS and HTML. You click in the pichu in the center of the screen, gain 'pat' points and spend them in the shop at the right. On the left, it's displayed the items you own and your stats.
You can play the game here: https://toderodavi.github.io/general-week02/
